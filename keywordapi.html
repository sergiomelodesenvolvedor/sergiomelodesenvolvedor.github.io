Affilyst is a personal tool designed to help affiliate marketers identify the best products to promote based on real data from the Google Ads API and AI-generated keyword suggestions. This tool is not public, not for sale, and is created solely for personal use in running real ad campaigns using Google Ads to promote affiliate products.

ðŸ§  How It Works

  1. The user uploads a .csv file containing product data (title, description, commission, score, and temperature).
  2. For each description, the tool uses ChatGPT (via the OpenAI API) to generate about 15 potential sales keywords.
  3. These keywords are sent in manual and controlled batches to the Google Ads API (Keyword Planner) to retrieve metrics such as CPC, search volume, and competition.
  4. The user analyzes the data and decides which products and keywords are most promising to invest in real paid campaigns on Google Ads.
