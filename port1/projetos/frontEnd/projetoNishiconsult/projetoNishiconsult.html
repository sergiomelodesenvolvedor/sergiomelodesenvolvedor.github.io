Prisioner Control

The Prisoner Control system aims to register detained individuals.
Each detainee record can be queried by common users or inserted and
edited by master users. Additionally, the master user has the ability to register new
users in the database.
For the implementation of this project, the following technologies were employed:
HTML, CSS, JavaScript, Node.js, and MySQL.
